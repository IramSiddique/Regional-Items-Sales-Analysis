# 📊 Region-Wise Sales Analysis of Cosmetics & Retail Products

## 🔹 Project Overview

This project analyzes region-wise sales data of cosmetics and other retail products to identify trends, performance, and key business insights.

## 🔹 Tools Used

* Microsoft Excel

## 🔹 Key Insights

* Top performing regions
* Best-selling products
* Revenue comparison across regions
* Sales trends and patterns

## 🔹 Dataset

The dataset contains region-wise sales information including product categories, revenue, and performance metrics.
